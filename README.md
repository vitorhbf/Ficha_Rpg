# Ficha de RPG

Aplicação web simples para registrar uma ficha de personagem com persistência local.

## Como abrir

1. Abra a pasta do projeto no navegador.
2. Clique em [index.html](index.html).

Ou rode um servidor local:

```bash
python -m http.server 3000
```

Depois acesse http://localhost:3000.
